# Example MDX Document

This is an example MDX document that showcases the features of MDX.

## Heading Levels

MDX supports multiple heading levels, which can be used to structure your document.

### Subheading

You can use subheadings to provide more detailed information.

## Lists

MDX supports both ordered and unordered lists.

### Unordered List

- Item 1
- Item 2
- Item 3

### Ordered List

1. Item 1
2. Item 2
3. Item 3

## Code Blocks

You can include code blocks in your MDX document using triple backticks.
